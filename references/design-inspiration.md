# Design Inspiration — 참조 패턴

유저가 "다양하게", "창의적으로", "심심하지 않게" 요청할 때 참조하는 디자인 영감 모음.
ADR(금지 규칙)과 달리 이건 **"이런 것도 할 수 있다"**는 가능성 확장용.
기본 생성에서는 ADR을 따르되, 유저가 다양성을 원하면 여기서 아이디어를 가져온다.

---

## 배경 기법

### 이미지 + 컬러 오버레이
- 실제 사진 위에 단색 오버레이 (70-80% opacity)
- 컬러는 primary/accent에서 선택
- 위에 흰 카드나 텍스트를 올림
- 효과: 시각적 풍성함 + 브랜드 컬러 유지

### 컬러 배경 + 장식 도형
- 전면 컬러 배경 (초록, 노랑, 보라 등) 위에 큰 원형/블롭 도형
- 도형은 같은 컬러 계열의 밝은/어두운 톤
- 모서리에 3D 에셋(이모지, 아이콘)을 배치해서 여백 채우기
- 효과: 에너지, 활기, 시선 집중

### 컬러 블록 분할
- 슬라이드를 2-3개의 컬러 블록으로 나눔
- 다크 블록 + 컬러 블록 + 화이트 블록 조합
- 각 블록 안에 서로 다른 콘텐츠 배치
- 효과: 비대칭, 역동적, 모던

---

## 콘텐츠 배치

### 에셋으로 여백 채우기
- 빈 공간에 3D 이모지를 rotate해서 장식으로 배치
- 카드 경계에서 삐져나오는 느낌 (overflow visible)
- 크기: 5-8cqw, opacity 100% (투명하게 하지 않음)
- 여백이 "비어 보이는" 문제를 해결

### 그룹핑은 배경 카드로
- 선(divider)으로 구분하는 대신, 라운드 카드 배경으로 그룹 경계 표현
- "main target group" 카드 안에 Seg 1, 2 / "sub target" 카드 안에 Seg 3
- 시각적으로 논리 구조가 한눈에 읽힘

### Pill Chip 적극 활용
- 해시태그: `#캠핑족 #홈파티 #미니아웃`
- 카테고리: `ATL` `BTL` `비주얼` `메시지`
- 기능 목록: 이모지 + 텍스트 pill
- 어두운 배경에는 밝은 pill, 밝은 배경에는 outlined pill

### 앱/제품 목업 배치
- 스크린샷 2-3장을 살짝 겹쳐서 깊이감
- 레이블 chip이 화살표와 함께 특정 부분을 가리킴
- 노란/보라/블루 배경 위에 올리면 제품이 돋보임

## 레이아웃 변형

### Needs → Solution 흐름
- 상단: 문제/니즈 카드 N개 (반투명 테두리)
- 하단: 솔루션 카드 N개 (흰 카드, 체크 아이콘)
- 점선으로 상하 연결
- 배경: 이미지+오버레이

### 2-Point 구조
- 좌우 2컬럼, 각각 Point 1/2
- 상단: 제목 + 설명 텍스트
- 하단: 카드 안에 태그/카테고리 chip 그룹
- 깔끔하면서 정보 밀도 높음

### 비대칭 블록 조합
- 좌상단: 다크 카드 (기능 목록)
- 좌하단: 라이트 영역 (텍스트)
- 우측 전체: 컬러 블록 + 제품 이미지
- 50:50이 아닌 불균형 → 더 역동적

## 색상 활용 원칙

- **단색 배경은 대담하게** — 연두, 노랑, 보라, 코랄을 두려워하지 마
- **어두운 배경 + 밝은 카드** — 대비가 콘텐츠를 살림
- **같은 색의 명암 변화** — 배경 초록 + 도형 진한초록 + 텍스트 흰색
- **회색만 쓰지 마** — 기본이 #F5F6FA면 무난하지만 심심. 유저가 원하면 컬러 적극 사용
